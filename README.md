# Machine Learning

This repository is a growing collection of machine learning concepts, algorithms, mathematical foundations, and hands-on implementations. It is intended to build a strong understanding of how machine learning models are developed, evaluated, and applied to solve real-world problems.

### Overview

The repository focuses on understanding the theoretical foundations of machine learning, implementing algorithms from scratch as well as using popular libraries, and applying them to practical datasets through well-documented Jupyter notebooks.

#### Unsupervised Learning
